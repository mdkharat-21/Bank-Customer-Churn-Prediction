# Bank Customer Churn Prediction

## Project Overview
This project predicts whether a customer will leave the bank or stay with the bank using Data Science techniques & Machine Learning algorithms.


## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset
Bank Customer Churn Dataset

## Machine Learning Workflow
1. Problem Statement
2. Data Collection
3. Exploratory Data Analysis
4. Feature Engineering
5. Feature Selection
6. Model Building
7. Hyperparameter Tuning
8. Model Evaluation

## Model Used
- Logistic Regression
- KNN
- Decision Tree
- Random Forest 

## Results

Several Machine learning algorithms were trained and evaluted incluing logistic regrssion,KNN,Decision tree,Random Forest
Among all models,Random Forest achieved the best performance with a train accuracy 87.34 and testing is 85.20
